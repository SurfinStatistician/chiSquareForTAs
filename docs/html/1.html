What Is Chi Square?

Chi Square is a a statistical procedure. It is sometimes called a test of independence. A chi square test measures the strength of association of two categorical variables.

Knowledge check: Can you run a chi square test with a continuous variable?

The overall purpose of this module set is to help you assess whether observations on two variables are independent of each other, using chi square test of independence.
