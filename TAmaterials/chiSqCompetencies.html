Chi Square: A Test of Independence

Chi Square is a a statistical procedure. It is sometimes called a test of independence.


Learning Objective 0: Assess whether observations on two variables are independent of each other, using chi square test of independence.

Learning Objective 1: Provide a basic description of a chi square test for two variables.

  • 1.1: Identify when it is appropriate to use a chi square test.
  • 1.2: Describe what a chi square analysis is testing.
  • 1.3: State the null hypothesis of a chi square test.
  • 1.4: State the alternative hypothesis of a chi square test.
  • 1.5: Interpret what a significant chi square test means, in terms of two variables.

Learning Objective 2: Interpret a frequency distribution table from a chi square test output.

  • Learning Objective 2.1: Differentiate between frequencies of cases and unique categories in a chi square output.
  • Learning Objective 2.2: Determine whether expected frequencies assumption is violated (e.g., interpret minimum number of expected values in a 2X2 contingency table for chi square).
  • Learning Objective 2.3: Write the formula for calculating degrees of freedom for chi-square test.

Learning Objective 3: Use tools for chi square test of independence appropriately

(e.g., Fisher's exact test, phi and Cramer's V)

  • Learning Objective 3.1: Differentiate between frequencies of cases and unique categories in a chi square output.
  • Learning Objective 3.2: Know the minimum number of expected values in a 2X2 contingency table for chi square.
  • Learning Objective 3.3: Write the formula for calculating degrees of freedom for chi-square test.
